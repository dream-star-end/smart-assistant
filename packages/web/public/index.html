还有什么想问的

OpenClaude 和官方 Claude.ai 有什么区别?

OpenClaude 是构建在 Claude API 之上的 AI 工作站。除了对话能力之外,还提供独立 Agent 容器、长期记忆、媒体生成、浏览器自动化、定时任务等官方网页版没有的工程化能力,适合把 AI 真正用到日常工作流里。

计费方式是按月订阅还是按量?

按量计费。充值后按实际 input / output token 扣账户余额,1 积分 = ¥1,精确到分。不用就不扣,余额永不过期。

Agent 模式跑在哪里?数据安全吗?

每位用户一个独立的 Docker 容器,文件系统、记忆、技能完全隔离。容器之间互不可见。所有数据仅用于服务你本人的 AI 助理,不会用于训练或对外分享。

支持哪些支付方式?

当前支持微信支付,通过虎皮椒聚合通道完成扫码支付。充值即时到账,可在账户中查看明细。

充错了 / 不想用了能退款吗?

未消费的余额可以申请退回原支付渠道,扣除支付通道手续费后退款。已消费的 token 不可退。

有手机 / 移动端吗?

Web 版完全响应式,在手机浏览器上即可使用全部功能,也可以"添加到主屏幕"作为 PWA 启动。
