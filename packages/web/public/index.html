还有什么想问的

和直接订 Claude Pro / Max 比有什么区别?

底座完全一样 —— 我们调用的就是官方 Claude Code Max 通道,模型是满血 Opus / Sonnet,不降智、不阉割上下文。差别在两点:① 多账号池轮询,你不会因为单账号 5 小时配额触顶而中断;② prompt cache 命中率 > 95%,token 账单比直接走 API 通常便宜一个数量级。按量付费,不订阅,不浪费。

"原生 Claude Code 底座"是什么意思?

市面上很多产品是"调 Claude API + 自己写一层壳"。我们直接用官方 Claude Code 作为底层 Agent 引擎 —— 工具系统、子 Agent 调度、文件操作、Bash、Grep、浏览器自动化全部原样保留。每位用户独立 Docker 容器跑一份完整的 Claude Code,行为与你本地装一份官方 CLI 完全一致。

计费方式是按月订阅还是按量?

按量计费。充值后按实际 input / output token 扣账户积分。充 ¥1 到账 100 积分,1 积分 ≈ $0.01 美元,精确到单 token。不用就不扣,积分永不过期。

Agent 模式跑在哪里?数据安全吗?

每位用户一个独立的 Docker 容器,文件系统、记忆、技能完全隔离。容器之间互不可见。所有数据仅用于服务你本人的 AI 助理,不会用于训练或对外分享。

支持哪些支付方式?

当前支持微信扫码支付,充值即时到账,可在账户中查看明细。

充错了 / 不想用了能退款吗?

充值后一律不支持退款,请按需充值。

有手机 / 移动端吗?

Web 版完全响应式,在手机浏览器上即可使用全部功能,也可以"添加到主屏幕"作为 PWA 启动。
